### About The Project
This is a restaurant website that let user browse restaurant list and search restaurant by name & type.
![screenshot1](https://i.ibb.co/M2W7DPT/2025-01-08-5-55-23.png)
![screenshot2](https://i.ibb.co/bHrSGsd/2025-01-08-5-55-44.png)

### Built With
- Node.js
- Express(Express-handlebars)
- Bootstrap
- Fontawesome
- Sequelize
- MySQL

### Function
- Browse: User can browse all restaurant in main page and see detail in detail page.
- Create : User can create a new restaurant into list.</br>

### How to Use:
- 1. clone git :https://github.com/clairehuang77777/restaurantList.git
- 2. cd your-repo
- 3. install npm, express, express-handlebar
- 4. "npm run dev" to run the server
- 5. Browse http://localhost:4001/ to see restaurant website
